# Jira N3 Dashboard Automation

Automação completa para triagem operacional de chamados N3 utilizando Google Apps Script, Jira REST API v3 e Power BI.

## Problema resolvido

Analistas N3 gastam tempo diariamente com:

- triagem manual
- priorização sem contexto
- risco de SLA vencido
- baixa previsibilidade operacional

## Solução

Painel automatizado que:

- sincroniza tickets a cada 10 minutos
- classifica chamados automaticamente
- envia briefing diário às 8h45
- destaca SLAs críticos
- consolida dados em dashboard

## Stack

- JavaScript
- Google Apps Script
- Jira REST API v3
- Google Sheets
- Power BI

## Funcionalidades

### Alertas Inteligentes

- SLA vencido
- Agir hoje
- Processo longo
- Zumbi
- Aguardando outro

### Automação

- atualização automática
- email diário
- classificação por prioridade
- dashboard operacional
